# 점프 투 FastAPI

> 개발환경 구성 
> - 개발환경 : VirtualBox + Ubuntu 22.04 Server
> - 백엔드 : FastAPI (uvicorn 실행)
> - 백엔드 프레임워크 : Pycharm 
> - 프론트엔드 : Svelte (Bootstrap 5.3.6 UI 기반)
> - 프론트엔드 프레임워크 : VS Code 
> - 데이터베이스 : SQLite
> - 배포용 GitHub Repository : khangte/fastapi_study

> $ uv venv
